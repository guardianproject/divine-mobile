# Category Emoji Selection Guide

All emoji are from [OpenMoji](https://openmoji.org/) (CC BY-SA 4.0).
Preview any emoji at: `https://openmoji.org/library/emoji-{UNICODE}/`

To swap any category's icon, download the desired OpenMoji SVG and replace the
corresponding file in this folder. The filename must match the category name
(lowercase), e.g., `music.svg`, `comedy.svg`.

## Alternative Options (featured categories)

The `options/` folder contains 5 alternatives per featured category.
To swap, copy the desired file over the active one:

```bash
cp options/music_headphones.svg music.svg
```
